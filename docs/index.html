Rivard Calculator (RIVARD)
Type: Associated Tool
Author: Seaver, A.
The Rivard Weights Calculation Program allows the user to convert a matrix of mid-year weights at age for a number of years to Jan-1 or other time of the year. The program uses the same algorithm as presently employed in the NFT Toolbox Virtual Population Analysis (VPA) program. This utility allows users of other assessment methods within the NFT Toolbox to access the Rivard methodology in a convenient manner.
